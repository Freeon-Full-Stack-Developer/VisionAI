1. Рабочий стол: Свернуть все открытые окна и показать рабочий стол.
2. Проводник: Открывает проводник Windows для просмотра файлов и папок.
3. Переключи окно: Переключение между открытыми окнами.
4. Скриншот: Делает снимок экрана.
5. Выключи звук: Выключает звук на компьютере.
6. Уменьши громкость: Уменьшает громкость звука.
7. Увеличь громкость: Увеличивает громкость звука.
8. Переключи раскладку: Переключает раскладку клавиатуры.
9. Меню Пуск: Открывает меню Пуск для доступа к приложениям и настройкам.
10. Сверни окно: Скрывает текущее окно.
11. Сверни все окна: Свернуть все открытые окна.
12. Центр уведомлений: Открывает центр уведомлений для просмотра уведомлений системы.
13. Создай виртуальный рабочий стол: Создает новый виртуальный рабочий стол для организации рабочего пространства.
14. Закрой виртуальный рабочий стол: Закрывает текущий виртуальный рабочий стол.
15. Перейди на рабочий стол слева: Переключается на виртуальный рабочий стол, расположенный слева.
16. Перейди на рабочий стол справа: Переключается на виртуальный рабочий стол, расположенный справа.
17. Сверни окна: Скрывает текущее окно.
18. Открой бар: Открывает игровую панель (Game Bar) для управления игровыми функциями.
19. Параметры: Открывает настройки Windows для изменения системных параметров.
20. Открой эмоции: Открывает панель эмодзи для вставки смайлов и символов.
21. Выбери: Нажимает клавишу Enter для выбора элемента.
22. Ниже: Перемещается вниз по списку.
23. Выше: Перемещается вверх по списку.
24. Левее: Перемещается влево.
25. Правее: Перемещается вправо.
